Información del Proyecto

Nuestra Misión

Air Quality es una plataforma que proporciona información en tiempo real sobre la calidad del aire en la zona de Guadalajara. Además, ofrece pronósticos para las próximas horas mediante el uso de inteligencia artificial y modelos de machine learning. Su objetivo es ayudar a las personas a tomar decisiones informadas sobre su salud y entorno, promoviendo una mayor conciencia sobre la calidad ambiental.

Nuestra Tecnología

Machine Learning

Modelo XGBoost optimizado para predicciones de alta precisión y análisis temporal.

ETL Pipeline

Apache Airflow (Astronomer) para orquestación de datos y reentrenamiento automatizado.

IA Generativa (RAG)

Integración de LangChain y LLMs para crear nuestro agente autónomo consultor.

Integración de APIs

Conexión ininterrumpida con OpenWeather para datos meteorológicos y de contaminación.

Infraestructura & Escalabilidad

Despliegue distribuido en Hugging Face y AWS RDS, diseñado para procesar y servir miles de consultas diariamente de forma eficiente y altamente escalable.
